# Rotation v0.29.1

This stable patch aligns Rotation's Homepage with the accepted warm, editorial
design language while preserving the specialist visual treatments used elsewhere.

## Highlights

- Restored the original warm palette, restrained shadows, and consistent 8px
  Homepage radii.
- Focused Home on the Focus Album, Player Rotation, and Library.
- Moved reflections, linguistic observations, and the role overview to a dedicated
  Insights page.
- Added direct Focus Album editing and replaced the large unbound notice with a
  compact Bindings attention indicator.
- Hardened API and Web container publication with explicit lowercase image names
  and deterministic tags generated without the failing metadata action.

## Images

- `ghcr.io/xyl0se/rotation-app-api:v0.29.1`
- `ghcr.io/xyl0se/rotation-app-web:v0.29.1`

Both containers must use the matching immutable tag. `v0.29.0` remains the exact
rollback target.
