# Insights Page

The Insights page is Rotation's quiet place for reflection and observation. It keeps
the Home page focused on listening while preserving the former Dashboard domains.

## Responsibilities

Insights combines:

- the next Reflection question and its Coach entry point;
- localized linguistic observations about the Library;
- a neutral overview of the roles currently represented.

It evaluates no Library and recommends no target role distribution. Album roles
describe relationships; they are not collection health metrics.

## Navigation

Insights is a first-class application page alongside Home, Bindings, Export, History,
and Settings. Home contains Focus Album, Player Rotation, and Library instead of a
Dashboard composition.

## Data Flow

The page loads the canonical Library from the API, evaluates Reflection and Insights
as projections, and persists role changes only after server confirmation. Archive
rediscovery continues to use the Archive Return Coach.

## Product Limit

Insights must remain editorial rather than analytical. It may explain patterns and
invite a decision, but must not introduce scores, warnings about role sizes, or a
concept of an optimal collection.
